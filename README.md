# CleanDiscord

A CSS file for removing unnecessary stuff from Discord.

## Changes

Removed:
- The Activity bar,
- Clan tags,
- Nitro nameplates,
- Labels on nitro button in settings,
- Labels on nitro button in DMs,
- Pulse gradient in VC,
- Avatar decorations,
- Profile effects,
- Custom banner in profiles.
